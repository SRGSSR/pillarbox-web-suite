Pillarbox Extensions

  • pillarbox-playlist

    Displays a list of available media items and handles playlist navigation and playback control within the Pillarbox player.

  • skip-button

    Adds a control to skip predefined video segments based on custom intervals triggered by the srgssr/interval event.

  • pillarbox-debug-panel

    Renders an overlay with internal debug data, including playback state, timing metrics, and live player information.

  • user-preferences

    Stores and retrieves user settings like volume, playback rate, text track, and audio track using localStorage.

  • big-replay-button

    Displays a prominent replay button when playback ends, reusing the standard play buttonâ€™s appearance and behavior.

  • brand-button

    Adds a configurable button to the control bar with a custom SVG icon and URL for linking to external destinations.

  • thumbnail-preview

    Displays timestamp-aligned thumbnails during timeline hover using a sprite sheet and optional scaling configuration.

  • card

    Displays a card component containing a title, duration, and a preview image.

  • chapters bar

    Displays a scrollable list of video chapters, allowing users to easily navigate through the content.
